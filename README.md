# altium-designer-pcb-libraries

Open-source PCB component libraries for Altium Designer.

## Libraries

| Library | Description |
|---------|-------------|
| `can_bus_ics` | CAN bus interface ICs |
| `generic_passives` | Generic passive component symbols |
| `generic_smd_chip_capacitors` | SMD chip capacitor footprints |
| `generic_smd_chip_resistors` | SMD chip resistor footprints |
| `ldos` | Low-dropout voltage regulators |
| `mcus` | Microcontrollers |
| `pin_headers` | Pin headers and connectors |
| `terminal_blocks` | Terminal block connectors |
| `usb_bus_ics` | USB interface ICs |
| `features` | Miscellaneous features |

## Usage

1. Download or clone this repository
2. In Altium Designer, go to **Preferences → Data Management → Installed Libraries**
3. Click **Install** and select the `.SchLib` and `.PcbLib` files you need
